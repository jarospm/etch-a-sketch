# Etch-a-Sketch

A browser-based sketchpad built as part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-etch-a-sketch) Foundations curriculum.

**[Live Demo](https://jarospm.github.io/etch-a-sketch/)**
